# BELMONTE_WASH_CARS
O Belmonte Wash Car’s é um sistema web de agendamento de serviços automotivos desenvolvido como Projeto Integrador do curso de Análise e Desenvolvimento de Sistemas (ENIAC – 2026).

🎯 Objetivo

    Digitalizar o processo de agendamento de lavagens automotivas, melhorando:

      Organização dos atendimentos
      Experiência do usuário
      Controle administrativo
      Comunicação entre cliente e empresa

⚙️ Funcionalidades
      🔐 Cadastro e login de usuários
      📅 Sistema de agendamento
      🧾 Painel do cliente
      🛠️ Painel administrativo
      📊 Controle de status (Pendente, Aprovado, Finalizado, Cancelado)
      👤 Gerenciamento de perfil
      
  👥 Perfis do Sistema
  🧑 Cliente
      Criar agendamentos
      Visualizar serviços
      Cancelar solicitações pendentes
      Atualizar dados pessoais
      
  🧑‍💼 Administrador
      Gerenciar todos os agendamentos
      Aprovar ou cancelar solicitações
      Finalizar serviços
      Visualizar estatísticas
      
  🛠️ Tecnologias Utilizadas
        🌐 HTML5
        🎨 CSS3
        ⚡ JavaScript
        🧠 PHP 8
        🗄️ MySQL
        🏗️ Arquitetura MVC
